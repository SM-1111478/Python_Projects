Temperature Advice
Start

Store the temperature in a variable called temperature.
If the temperature is less than 10:
Print "It's cold outside. Wear a jacket!"
Else if the grade is between 10 and 25 (inclusive):
Print "It's cold outside. Wear a jacket!"
